COLT - Code Orchestra Livecoding Tool

Incredibly Fast AS3 Compiler

COLT is a small program to test and build mobile, web and desktop-based applications in real time. Developers call this “live coding” or “live development”.

Connecting COLT to any editor, a programmer does not waste time compiling applications multiple times and searching for errors.

Timesaver

Super fast incremental compilation - (~200 ms). No loss during application recovery after compilation — it works just like before. Apps that are already on devices don't need to be reinstalled after compilation.

Mobile Development

COLT supports software development for iOS and Android.

In order to put an application together you only need to set up assembly of a mobile app and click the “Run” button.

No Restrictions

COLT is a separate tool.

No particular IDE is required. You just save a code in your editor and then it's delivered into a working app. Support of all win/mac/linux platforms.
