C.O.L.T. — Code Orchestra Livecoding Tool

  • JavaScript
  • ActionScript

COLT is a small program to test and build mobile, web and desktop-based applications in real time.

Developers call this “livecoding” or “live development”.
Connect COLT to any editor, keep calm and code on!

[LiveCodeUpdateListener]
function liveCodeListener(){
  trace("class code  updated");
}

[LiveCodeUpdateListener(method="initInterface")]
function liveCodeListener(){
  trace("method initInterface code updated");
  clearInterface();
  initInterface();
  trace("class code  updated");
}

[Embed(source ="icon.png")]
private Class imageClass;

[LiveAssetUpdateListener(field="imageClass")]
function iconChanged(){
  trace("image icon.png changed");
  setIcon(new imageClass())
}

private Class imageClass2;

function resourcesChanged(){
  trace("image.png or icon2.png changed")

Timesaver

Super fast incremental compilation - (~200 ms). No loss during application recovery after compilation — it works just like before. Apps that are already on devices don't need to be reinstalled after compilation.

Use the Editor of Your Choice

COLT is a separate tool. No particular IDE is required. Just save a code in your editor — and all changes are delivered into the working application.

They already achieve the impossible: live coding in AS3… It will become the most useful tool for AS3 developer!

Teamwork

An app in a live session mode is network-available. COLT starts a web server able to deliver updates even if the app has been launched on another computer somewhere in the office.

And much more...

  • COLT can be used to develop for iOS and Android.
  • COLT supports unsafe code like infinite loops and recursions.
