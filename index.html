COLT is a small program to test and build mobile, web and desktop-based applications in real time.

Compatibility with Popular Technologies

$(window).liveUpdate(function (e, files) {
  console.log("files " + files.join(", ") + " were changed");
});

$(window).codeUpdate(function(e, fncs){
  $.each(fncs, function(){
    console.log("function was changed: " + this);
  });
});

$(window).imageUpdate(function (e, images) {
  $.each(images, function(){
    console.log($(this).attr("src") + " file was changed");
    $(this).imageUpdate();
  });
  // or  refresh all images
  $(window).imageUpdate();
});

$(window).cssUpdate(function (e, stylesFiles) {
  $.each(stylesFiles, function(){
    console.log(this + " css file was changed");
  })
  // refresh styles
  $(window).cssUpdate()
});

The Web App / Page Gets Instantly Updated

  • Partial updates — refresh a page without full reload of css, images.
  • Smart live reload — when you change the HTML and other resources, the page is automatically reloaded.

Use the Editor of Your Choice

COLT is a separate tool. No particular IDE is required. Just save a code in your editor — and all changes are delivered into the working application.
